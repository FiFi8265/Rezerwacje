⚠️ Konfiguracja Firebase - TYLKO DLA ADMINISTRATORA

UWAGA: Ten formularz jest widoczny tylko dlatego, że nie uzupełniłeś danych Firebase w kodzie. Po uzupełnieniu danych w pliku HTML, ten formularz zniknie dla wszystkich użytkowników.

Instrukcja:

  1. Otwórz plik HTML w edytorze tekstu (np. Notepad++, VS Code)
  2. Znajdź sekcję z kodem:
    const firebaseConfig = {
  3. Zastąp wszystkie wartości "TUTAJ-WKLEJ-..." swoimi danymi z Firebase
  4. Zapisz plik i wgraj ponownie na GitHub

Twoje dane Firebase:

const firebaseConfig = { apiKey: "twój-api-key-tutaj", authDomain: "twój-auth-domain.firebaseapp.com", projectId: "twój-project-id", storageBucket: "twój-storage-bucket.appspot.com", messagingSenderId: "twój-sender-id", appId: "twój-app-id" };
